# 🤖 Desktop-AI-Organizer

> A powerful Windows desktop app that analyzes, organizes, and manages your desktop files with AI-powered insights and power tools (Duplicate Finder, Batch Rename, File Shredder, Bulk Cleanup, System Info)

## ✨ Features

- Core functionality as described above
- Built with modern technologies
- Easy to set up and use

## 💻 Installation

```bash
git clone https://github.com/zougar99/Desktop-AI-Organizer.git
cd Desktop-AI-Organizer
```
