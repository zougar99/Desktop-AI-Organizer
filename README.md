# 🤖 Desktop-AI-Organizer

> A powerful Windows desktop app that analyzes, organizes, and manages your desktop files with AI-powered insights and power tools (Duplicate Finder, Batch Rename, File Shredder, Bulk Cleanup, System Info)

---

## ✨ Features

- ✔ **AI-powered analysis and automation**
- ✔ **Smart predictions and insights**
- ✔ **Natural language processing**
- ✔ Lightweight and optimized
- ✔ Cross-platform compatibility
- ✔ Regular updates

---

## 💻 Tech Stack

| Component | Technology |
|-----------|-----------|
| Language | Python |
| UI | CustomTkinter / PyQt |
| Platform | Windows / Linux / macOS |

---

## 🚀 Installation

```bash
git clone https://github.com/zougar99/Desktop-AI-Organizer.git
cd Desktop-AI-Organizer
pip install -r requirements.txt
```

---

## 🧰 Usage

```bash
python main.py
```
